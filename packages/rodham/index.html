rodham

Network of emails - random walk clustering (color).

rodham is an R package that includes a dataset of Hillary Rodham Clinton's personal emails as well as function to download their entire content from the Wall Street Journal and read it into R. rodham is available on CRAN.

Install

install.packages("rodham") # stable version from CRAN
devtools::install_github("JohnCoene/rodham") # development version
